# setup

- install browser extension, load unpacked `./bex`
- run server `cd ./server && pnpm i && pnpm start`

# usage

- use 'save image' right click menu
- all pins go into `./pins`
- use whatever tool to browse pins
- i use https://blyt.net/phxslides/
